# Firegate – Inline Network Security (Plug & Play)

Firegate is a hardware cybersecurity device that sits between your modem and router and blocks threats in real time.

## Why this exists

Most home and small business networks are constantly exposed to:

- Unsolicited connection attempts
- Malicious domains and phishing
- Botnet scanning and probing
- Hidden traffic from IoT devices

These threats are usually invisible to the user.

## What Firegate does

- Inline IPS (real-time traffic inspection using Suricata)
- Blocks malicious domains and connections
- Enforces DNS filtering (prevents bypass)
- Detects suspicious behaviour at network level
- Works without cloud dependency
- No data collection

## Architecture

Firegate operates inline:

Modem → Firegate → Router → Devices

- Transparent inspection (no configuration required)
- No changes needed on router or devices
- Plug-and-play deployment
